Аутентификацию - процесс когда мы узнаём нашего пользователя. Процедура узнавания.
Авторизация - это раздача прав (админ, пользователь, модератор).

Реализуем аутентификацию используя JWT-token.
(JSON Web Token)
Как работает:
    - Приходит пользователь, вводит логин и пароль, мы смотрим связку в БД. Если она правильная,
    то генерируем некий JWT-token и выдаём его пользователю.
    - Далее когда клиент делает запрос к нашему API он должен каждый раз показывать данный токен.
    - Токен можно выдавать на время.

Добавляем пакет для создания JWT токенов:
    go get github.com/auth0/go-jwt-middleware

Добавляем пакет для работы с JWT токенами (контролирование):
    go get github.com/dgrijalva/jwt-go
